To add a Note to your File Cabinet, You must first load or create a File Cabinet. Then You must add a Main Folder and a Sub Folder. Now You can add a Note

To create a File Cabinet

  1. Click on File and then select Create File Cabinet
  2. Enter a name and click Add New File Cabinet.
  3. Enter a name for the path and click Save.

To load a File Cabinet

  1. Click on File and then select Load File Cabinet.
  2. Navigate to the file where you have your File Cabinet.
  3. Click on it and click open.

To change the File Cabinet name.

  1. Hold down shift and click the File Cabinet Name.
  2. Fill in the form and click Rename File Cabinet.

To add a Main Folder

  1. Click on the File Cabinet name,
    and a Main Folder section will pop up.
  2. Click on the small icon beside the word Main Folder.
  3. This will bring up a form.
  4. Enter a name for your new Main Folder and click Add Main Folder.

To change the Main Folder name.

  1. Hold down shift and click the Main Folder Name.
  2. Fill in the form and click Rename Main Folder.

To add a Sub Folder

  1. Click on a main folder tab, and a Sub Folder section will pop up.
  2. Click the small icon beside the word Sub Folder.
  3. This will bring up a form.
  4. Enter a Name for your new Sub Folder and click Add Sub Folder.

To change the Sub Folder name.

  1. Hold down shift and click the Sub Folder Name.
  2. Fill in the form and click Rename Sub Folder.

To add a Note

  1. Click a Sub Folder tab, and a note section will pop up.
  2. Click the small icon beside the word Note.
  3. This will bring up a form for your note. Enter your Note and click Add Note.

To add a Image to a Note.

  1. To add an image to a Note, hold down alt and click on a note, the file picker will promt you for a path.
  2. Find your picture and select it and click open.
  3. To see the picture that belong's to the Note, Just click on the Note and the picture will pop up.
  4. To dismiss the picture just click the picture and it will go away.
  5. To remove an image path from a note dimiss the picture by clicking on it, it will disapear, and the hold down shift and click on the Note.
  6. To remove an image path that no longer exists, hold down shift and click on the Note.

To enter Edit and Delete Mode

  1. Click on Mode and select Edit and Delete

To exit Edit and Delete Mode

  1. Click on Mode and select Read and Write

Delete a Main Folder, Sub Folder, or Note

  1. First enter Edit and delete Mode.
  2. See above.
  3. To delete Main Folder or Sub Folder hold down control key and click the trash can to the right of the folder name you want to delete.
  4. Deleting a Main Folder or Sub Folder will delete everything under it.
  5. To delete a note click on the black trash can above the note.

To move a note up or down the screen

  1. First enter Edit and Delete Mode.
  2. See above.
  3. Click the arrow's beside the trash can.
  4. Click the up arrow to move up.
  5. Click the down arrow to move down.

To edit a note.

  1. First enter Edit and Delete Mode.
  2. See above.
  3. Click the pencil icon close the trash can.
  4. Edit Your Note in the form.
  5. Click Save Edit button.

To Close a File Cabinet.

  1. Select a file cabinet by clicking on it.
  2. Click File and select Close Selected Cabinet.

To Close All File Cabinet's.

  1. Click File and select Close All File Cabinet's

To auto load file cabinet's

  1. Click Settings and click Start Up Setting's Form.
  2. Check the box beside Auto Load File Cabinet's.
  3. Click the Add File To Auto Load List.
  4. Add as many file's as you like.
  5. After you add all your files, click the Save Settings button.
  6. To remove a file from the auto load list,
    enter Edit and Delete mode, hold down the control key and click the
    red trash can beside it and click the Save Settings button.

Start Up Settings

  1. Click Settings and click Start Up Setting's Form.
  2. Pick your theme.
  3. Pick your font.
  4. Add as many file's as you like, to the auto load list.
  5. Check the autoload box.
  6. After you make your selection's click the Save Settings button.

Factory Reset

  1. Click Settings and click Start Up Setting's Form.
  2. Click Factory reset
  3. This will set setting's back to default values
